# o0_o.windows

Windows utilities and action plugin base classes for Ansible.

## Documentation

Full documentation is available at:
https://o0-o.github.io/ansible-collection-windows/

## Installation

```bash
ansible-galaxy collection install o0_o.core
```

> **Note:** This collection is part of the `o0_o` namespace and requires
> other collections in the namespace to function. Install the full namespace
> via `o0_o.core` as shown above.
